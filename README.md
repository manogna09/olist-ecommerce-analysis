
# Olist E-commerce Data Analysis

This project analyzes real-world e-commerce data from the Olist dataset using Python, pandas, and matplotlib.

## Features
- Monthly sales analysis
- Top and least ordered product categories
- Review score distribution
- Delivery time statistics
- Orders by customer state
- Payment method analysis

## How to Run
1. Download the Olist dataset CSVs from Kaggle and place them in the `data/` directory.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the analysis script:
   ```bash
   python analysis/eda.py
   ```
